# 💰 Plataforma de Gestão de Despesas Pessoais

## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de um **protótipo funcional de um aplicativo mobile** voltado para o controle de despesas pessoais.

A aplicação foi criada com foco em **simplicidade, usabilidade e acessibilidade**, permitindo que usuários registrem seus gastos e receitas de forma rápida, visualizem seu saldo e acompanhem seus hábitos financeiros.

Diferente de um protótipo estático, este projeto simula um sistema real utilizando **HTML, CSS e JavaScript**, com persistência de dados via **localStorage**.

---

## 🎯 Objetivo

Validar a navegação, fluxos e experiência do usuário com base em personas e jornadas previamente definidas, além de simular funcionalidades reais de um sistema financeiro.

---

## 👤 Personas

### João (Motorista de aplicativo)

* Precisa registrar gastos rapidamente
* Possui rotina dinâmica
* Busca simplicidade

### Laura (Freelancer)

* Precisa controlar receitas e despesas
* Deseja visualizar melhor seu fluxo financeiro
* Busca organização e planejamento

## 🚀 Funcionalidades

### 💸 Gestão de despesas

* Adicionar gastos
* Editar gastos
* Remover gastos

### 💰 Gestão de receitas

* Adicionar receitas

### 📊 Dashboard

* Visualização de:

  * Total de receitas
  * Total de gastos
  * Saldo atual

### 📈 Relatórios

* Agrupamento por categoria
* Gráfico de despesas (canvas)

### 💾 Persistência

* Dados armazenados localmente no navegador (localStorage)

O sistema permite simular os principais fluxos de um aplicativo real:

* Cadastro de gastos e receitas

* Visualização de resumo financeiro
